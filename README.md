# AI Tooling

Global tooling workspace. Each toolset is isolated under `tools/<context>/` with its own README and Makefile.

## Toolsets

- `tools/prompt-analysis` - Analyze Claude/OpenCode sessions, generate reports, stats, and chunks.
- `tools/opencode-token-report` - Token usage report by model from local opencode session data.

## Usage

Enter a toolset directory and follow its README.

```bash
make all
```
